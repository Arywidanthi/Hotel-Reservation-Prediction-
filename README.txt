Project Title: Hotel Reservation Prediction
Objective:
Created a machine learning classification project to use the information provided to predict if the customer will cancel the reservation or not. Finding the classification model that is appropriate for the data using cross-validation method and get accuracy of 77%.
Data Source: https://www.kaggle.com/datasets/ahsan81/hotel-reservations-classification-dataset
Deployment Link: https://huggingface.co/spaces/arywidanthi/hotel_reservation_prediction
